# 📚 Study Tracker (学習記録・分析アプリ)

自分の学習時間を記録し、グラフで可視化することでモチベーションを維持するためのWebアプリケーションです。
インターン応募用のポートフォリオとして作成しました。

## 🛠 使用技術
* **Backend:** Python 3.12, Django 4.2
* **Frontend:** HTML5, Bootstrap 5, Chart.js (グラフ描画)
* **Infrastructure:** Render (Web/DB), PostgreSQL
* **Other:** Git/GitHub

## 🔥 こだわったポイント
1. **データの可視化**
   Chart.jsを用いて「日別の学習推移（棒グラフ）」と「科目のバランス（円グラフ）」を実装しました。
2. **マルチユーザー対応**
   ログイン認証機能を実装し、ユーザーごとにデータが完全に分離されるように設計しました。
3. **直感的なUI**
   Bootstrapを使用し、スマホからでも入力をしやすいレスポンシブデザインにしました。
